# 主要功能:
1.带进度条的webView.<br/>
2.图片上传,上传的时候可以选择图片,也可以相机拍照.<br/>
3.文件下载,下载完成后可直接点击下载通知打开,其中若是下载的apk,可直接点击下载通知进行安装.<br/>
4.在html中有打开app的按钮的话,如果本地已经安装了目标app,可直接打开app,在源码中我有调用支付宝支付的demo,可下载源码进行体验.<br/>
